# MOOM Member UI Kit

LINE LIFF mini-app for members. Mobile-first, 390px canvas, gamified.

## Screens in the kit
- **Home** — greeting, momentum card (XP + tier), today's plan, quick actions
- **Schedule** — weekly class grid, booking chips
- **Check-in** — QR scan with animated scan-line
- **Rewards** — XP ledger + badges
- **Profile** — tier, streak, stats

## Components
`MomentumCard`, `TierBadge`, `XPBar`, `StreakFlame`, `ClassCard`, `BottomNav`, `TopBar`, `QuickAction`, `BadgeGrid`, `QRScanner`.

Demo: `index.html` — click-through prototype with screen tabs + tweakable tier.
