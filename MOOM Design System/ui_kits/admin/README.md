# MOOM Admin UI Kit

Desktop dashboard for gym owners, managers, trainers, front desk. 1280+ canvas, information-dense.

## Screens
- **Dashboard** — KPI tiles, schedule timeline, needs-attention, AI daily briefing
- **Members** — searchable table with risk scoring + tier
- **Schedule** — class list + roster
- **Finance** — revenue breakdown + transactions

## Components
`Sidebar`, `TopHeader`, `KpiCard`, `MembersTable`, `ClassSchedule`, `AttentionList`, `AIBrief`, `RevenueChart`.

Demo: `index.html` — click-through between the four sections.
